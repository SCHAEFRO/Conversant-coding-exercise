Conversant coding exercise. 

Contains the follwoing files. 


1. DataAnalysis.java - java source files

2. DataAnalysis.class - Java compile class file. 

3. data.Montoya.csv - txt file containing data for the DataAnalysis program. 

4. exercise.jar - executable jar file to run program. 



Instructions for running DataAnalysis program. 


The DataAnalysis.java class was compiled with Java 8 u91 (64 bit). 

1. Copy the exercise.jar file to any temp directory. 

2. Make sure your system path can execute the java.exe command. 

3. Execute the following command to run the program. 

  >java -jar exercise.jar
  
  the output of the program will be written to the console where the program is run.
